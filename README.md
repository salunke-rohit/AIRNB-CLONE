"# Airbnb Clone" 

WonderLust is a full-stack web application that allows users to create, view, update, and delete property listings.
It is built using Node.js, Express.js, MongoDB, Mongoose, and EJS, following MVC architecture.

Features

 View all listings
 Add new listings
 Edit existing listings
 Delete listings
 Image support with default fallback
 MongoDB database integration
 RESTful routing
 MVC project structure
 Server-side rendering using EJS templates
